自定义安装

  1. 双击安装包

    Install Now以管理员权限安装Python，勾选添加到环境变量

  2. 可选功能

    • Documentation: 安装Python文档文件。选择这个选项会在你的Python安装目录中包含Python的官方文档，方便你离线查阅Python的语法、库、模块等信息。如果你经常需要查看Python文档，或者没有稳定的网络环境，安装文档会很有帮助。
    • pip: pip是Python的包管理工具，是Python开发中非常重要的工具，可以用来下载和安装其他Python包。
    • tcl/tk and IDLE: 安装Tkinter和IDLE开发环境。Tkinter是 Python的标准GUI库，基于Tcl/Tk，可以用来创建简单的桌面应用程序。IDLE是Python的集成开发环境(IDE)，提供了一个简单的文本编辑器和交互式解释器，适合初学者编写和调试Python代码。
    • Python test suite: 安装标准库测试套件。Python的标准库包含了大量的测试用例，用于验证Python解释器和标准库的功能是否正常。通常用于开发人员进行Python解释器的测试和调试，对于普通用户来说，这个选项可以不选。
    • py launcher for all users: py launcher是一个命令行工具，可以让你在命令行中使用 py 命令来启动 Python 程序，而不需要输入完整的Python可执行文件路径。会安装到系统目录中，所有用户都可以使用。安装这个选项需要管理员权限(elevation)。

  3. 高级选项

    • Install for all users: 为所有用户安装。选择这个选项会将Python安装在系统目录下(通常是 C:\Program Files\Python版本号)，这样所有用户都可以访问和使用Python。如果不选，则只安装在当前用户的目录下(通常是 C:\Users\用户名\AppData\Local\Programs\Python\Python版本号)，只有当前用户可以访问。
    • Associate files with Python(requires the py launcher): 将文件关联到Python(需要py启动器)。选择这个选项会将.py文件关联到Python，这样你双击.py文件时，系统会使用Python来执行该文件。这个选项需要前面勾选安装py启动器，因为py启动器负责解析和执行.py文件。
    • Create shortcuts for installed applications: 为安装的应用程序创建快捷方式。选择这个选项会在开始菜单中为Python的应用程序创建快捷方式，方便快速启动应用程序。
    • Add Python to environment variables: 将Python添加到环境变量。选择这个选项会将Python的可执行文件路径添加到系统的环境变量中(通常是 Path 变量)，这样你可以在命令行中直接使用python命令来启动Python解释器，而不需要输入完整的路径。
    • Precompile standard library: 预编译标准库。选择这个选项会在安装过程中将Python的标准库模块预先编译成字节码文件(.pyc 文件)，这样在运行Python程序时，可以加快标准库模块的加载速度，提高程序的启动速度。
    • Download debugging symbols: 下载调试符号。调试符号包含了Python解释器和库的调试信息，选择这个选项会下载调试符号文件。调试符号主要用于开发人员进行调试和分析Python程序的性能问题，对于普通用户来说，这个选项可以不选。
    • Download debug binaries(requires VS 2015 or later): 下载调试二进制文件(需要 Visual Studio 2015 或更高版本)。调试二进制文件是Python解释器和库的调试版本，包含了调试信息和符号。选择这个选项会下载调试二进制文件，但需要你安装了Visual Studio 2015或更高版本的开发环境。调试二进制文件同样主要用于开发人员进行调试和分析Python程序的问题，对于普通用户来说，这个选项也可以不选。
    • Customize install location: 自定义安装位置
